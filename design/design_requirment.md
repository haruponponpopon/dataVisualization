# 層グラフのデータフォーマット案

- jsファイルで作成する．
- データフォーマット

```javaScript
ShopData = {
        genre_num: [ジャンル総数],
        min_date: [日付最小値],
        max_date: [日付最大値],
        genres: [ジャンル1, ジャンル2, ジャンル3...], 
        dist: [
                    [(2011年1月のデータ)ジャンル1, ジャンル2, ジャンル3...],
                    [(2011年2月のデータ)ジャンル1, ジャンル2, ジャンル3...],
                        ...
                    [(2021年10月のデータ)ジャンル1, ジャンル2, ジャンル3...]
                ]
    }
```

datasは各ジャンルにおける累計の店舗数
